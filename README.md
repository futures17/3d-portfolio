# Ayush Pathak | 3D Portfolio & Java Developer

Welcome to my official 3D Portfolio! This project is a high-performance, visually striking web application built with React, Vite, and Framer Motion.

## 🚀 Quick Start

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
```

## 🌐 Deployment (Netlify)

To host this project on Netlify:

1. **Build the project**: Run `npm run build`.
2. **Locate the `dist` folder**: This is your production-ready website.
3. **Deploy via Netlify Drop**:
    - Zip the contents of the `dist` folder.
    - Drag and drop the zip file onto [Netlify Drop](https://app.netlify.com/drop).
4. **Deploy via GitHub**:
    - Connect your repository to Netlify.
    - Set Build Command: `npm run build`.
    - Set Publish Directory: `dist`.

## ✨ Features
- **3D Interaction Layer**: Premium custom cursor with inversion and glow effects.
- **Dynamic Content**: Auto-sliding testimonial reviews and tilted marquee.
- **Video Backgrounds**: High-fidelity HLS and MP4 background integration.
- **SEO Optimized**: Fully configured meta tags for social sharing and search engines.
- **Responsive Design**: Seamless experience across mobile, tablet, and desktop.
